🚀 ML & DL Production Pipelines: Titanic & House Prices
Данный репозиторий представляет собой модульный, конфигурируемый и полностью воспроизводимый фреймворк для решения задач табличного машинного обучения и глубокого обучения (Deep Learning).

Реализованы два пайплайна под две классические задачи:

Titanic (Бинарная классификация).

House Prices (Регрессия с логарифмированием таргета).

Каждая задача имеет как Classical ML реализацию (бустинги, стекинг), так и Deep Learning реализацию (PyTorch, Entity Embeddings).

🌟 Ключевые особенности (Features)
Управление через OmegaConf: Вся логика экспериментов (архитектура, препроцессинг, гиперпараметры) вынесена в конфигурационные файлы. Никакого хардкода в пайплайнах.

Продвинутая импутация пропусков: Реализованы кастомные классы AdvancedNumericalImputer и AdvancedCategoricalImputer, поддерживающие математические сдвиги ([MIN] - Const) и предсказание пропусков через ML (pseudo_labeling на базе IterativeImputer).

Entity Embeddings (DL): Категориальные признаки обрабатываются через динамические слои nn.Embedding, что позволяет нейросети самостоятельно выучивать связи между категориями.

OOM-Safe Инференс: Предсказания на больших объемах тестовых данных происходят батчами через DataLoader, защищая память от переполнения (Out-Of-Memory).

100% Воспроизводимость: Жесткая фиксация seed во всех библиотеках (NumPy, Python, PyTorch, CUDA, воркеры DataLoader'ов).

Stacking Factory (ML): Динамическая сборка многоуровневых ансамблей из CatBoost, XGBoost, LightGBM, Random Forest и линейных моделей в пару кликов.
